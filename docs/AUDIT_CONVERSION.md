# Audit Conversion Landing Page RA Bâtiment

**Date** : 5 Janvier 2026
**Branche** : `feature/mobile-menu-sticky-cta`
**Objectif** : Optimiser le taux de conversion des prospects visitant la landing page

---

## Executive Summary

La landing page presente une identite visuelle premium (noir/or) coherente avec le positionnement haut de gamme. Cependant, plusieurs problemes critiques nuisent a la conversion :

1. **Faux logos partenaires** (Bouygues, Vinci...) - risque de credibilite
2. **Formulaire trop long** (5 champs) - friction elevee
3. **Manque de RGPD** - non-conformite legale
4. **Photos stock** au lieu de vraies realisations

**Note globale : 7.05/10**

## Analyse Section par Section

### 1. HEADER (7/10)

**Points forts :**
- Logo distinctif avec initiales "RA" dorees
- Navigation claire (4 liens)
- CTA "Consultation Privee" visible sur desktop
- Menu mobile hamburger avec animation

**Problemes :**
- CTA disparait sur mobile (< 1024px)
- "Consultation Privee" est vague
- Pas de telephone dans le header desktop

**Recommandations :**
| Prio | Action |
|------|--------|
| P2 | Ajouter telephone cliquable dans header desktop |
| P2 | Reformuler CTA : "Devis Gratuit en 24h" |
| P3 | Badge "Reponse sous 24h" a cote du CTA |

---

### 2. HERO SECTION (8/10)

**Points forts :**
- Image de fond luxueuse avec overlay sombre
- Badge credibilite "Artisan d'Excellence depuis 15 ans"
- Titre percutant "L'Art de Batir le Prestige"
- 2 CTAs visibles (Devis + Telephone)
- Stats flottantes (15+ Annees, 500+ Projets, 100% Satisfaction)
- Animation Framer Motion soignee

**Problemes :**
- Stats masquees sous 1280px (majorite mobile)
- Sous-titre manque de benefices concrets
- Pas de video explicative
- Pas d'urgence/scarcite

**Recommandations :**
| Prio | Action |
|------|--------|
| P1 | Afficher stats sur mobile (sous le titre) |
| P1 | Proposition de valeur concrete : "Renovation cle en main - Devis sous 24h - Garantie Decennale" |
| P2 | Element d'urgence : "Prochaines disponibilites : Mars 2026" |
| P3 | Video testimoniale courte (30-60s) |

---

### 3. TRUST BADGES (8.5/10)

**Points forts :**
- 4 badges pertinents et design coherent
- Position strategique post-hero

**Problemes :**
- Badges generiques, pas verifiables
- Manque "Devis Gratuit"

**Recommandations :**
| Prio | Action |
|------|--------|
| P2 | Ajouter "Devis Gratuit & Sans Engagement" |
| P3 | Lier certifications a des documents |
| P3 | Badge "Assurance RC Pro" |

---

### 4. SERVICES (7.5/10)

**Points forts :**
- 6 services clairs avec images de qualite
- Numerotation elegante (01, 02...)
- Animation au scroll

**Problemes :**
- 6 cartes = trop de choix (dilution attention)
- Liens vers #contact (pas de pages dediees)
- Pas de fourchette de prix

**Recommandations :**
| Prio | Action |
|------|--------|
| P2 | Badge "Plus demande" sur 2-3 services phares |
| P2 | Fourchette de prix ou "A partir de" |
| P3 | Pages dediees par service |

---

### 5. PORTFOLIO (7/10)

**Points forts :**
- 6 realisations avec localisation
- Layout avec image featured
- Effet zoom au hover

**Problemes :**
- Pas de filtre par categorie
- Manque budget/surface/duree
- Images Unsplash (pas de vrais projets)
- Pas de CTA "Voir plus"

**Recommandations :**
| Prio | Action |
|------|--------|
| P1 | VRAIES photos de chantiers RA Bâtiment |
| P2 | Donnees concretes : "45m2 - 3 semaines - 25k EUR" |
| P2 | CTA "Voir toutes nos realisations" |
| P3 | Filtrage par type de projet |

---

### 6. TEMOIGNAGES (6.5/10)

**Points forts :**
- 3 temoignages 5 etoiles
- Noms + localisation + type de projet

**Problemes :**
- Seulement 3 temoignages (volume faible)
- Pas de photos clients (credibilite -40%)
- Pas de dates
- Pas de lien Google Reviews

**Recommandations :**
| Prio | Action |
|------|--------|
| P1 | Integrer vrais avis Google avec widget |
| P1 | Photos avant/apres des projets |
| P2 | Date de l'avis visible |
| P2 | Lien vers profil Google : "Voir nos X avis (4.9/5)" |
| P3 | Temoignages video courts |

---

### 7. TRUSTED BY / PARTENAIRES (5/10) - CRITIQUE

**Problemes MAJEURS :**
- Logos de GRANDS GROUPES (Bouygues, Vinci, Eiffage, Nexity)
- Improbable qu'un artisan individuel soit partenaire du CAC 40
- Risque de credibilite si detecte
- Risque legal potentiel

**Recommandations :**
| Prio | Action |
|------|--------|
| P1 CRITIQUE | Supprimer OU remplacer par vrais partenaires |
| P1 | Alternatives : Cedeo, Point.P, BigMat, labels RGE/Qualibat |

---

### 8. FORMULAIRE DE CONTACT (6/10)

**Points forts :**
- Champs logiques
- Feedback visuel loading/success/error

**Problemes :**
- 5 champs = trop de friction
- Pas de checkbox RGPD (obligatoire France)
- CTA generique "Envoyer ma demande"
- Pas de rappel benefices

**Recommandations :**
| Prio | Action |
|------|--------|
| P1 | Reduire a 3 champs : Nom, Telephone, Message |
| P1 | Checkbox RGPD obligatoire |
| P1 | CTA : "Recevoir mon devis gratuit sous 24h" |
| P2 | Sous le form : "Reponse garantie sous 24h - Sans engagement" |
| P3 | Integration Calendly pour RDV direct |

---

### 9. MOBILE STICKY CTA (8/10)

**Points forts :**
- Visible uniquement mobile (< 768px)
- 2 actions claires : Appeler + Devis
- Design discret mais accessible

**Problemes :**
- Peut masquer contenu en bas
- Pas de micro-animation

**Recommandations :**
| Prio | Action |
|------|--------|
| P3 | Animation subtile apres 5s de scroll |
| P3 | Afficher le numero au lieu de "Appeler" |

---

### 10. FOOTER (7/10)

**Points forts :**
- Logo + liens essentiels
- Mentions legales
- Copyright a jour

**Problemes :**
- Pas d'horaires visibles
- SIRET non affiche (obligatoire)
- Pas de reseaux sociaux
- Pas de zone d'intervention

**Recommandations :**
| Prio | Action |
|------|--------|
| P1 | Afficher SIRET (obligation legale) |
| P2 | Horaires : "Lun-Sam 8h-19h" |
| P2 | Zone : "Ile-de-France : Paris, 92, 93, 94, 95" |
| P3 | Icones reseaux sociaux |
