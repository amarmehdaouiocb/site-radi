# Project Instructions for Claude Code

## Deployment

- **Netlify auto-deploy** : chaque push/merge sur `master` déclenche un déploiement automatique
- Pas besoin de redéployer manuellement après un merge

## Stack

- **Framework** : Next.js 16 (App Router)
- **Styling** : Tailwind CSS + CSS custom (gold.css)
- **Email** : Resend API
- **Hébergement** : Netlify
- **Domaine** : ra-batiment.fr

## Conventions

- Langue du site : Français
- Accents obligatoires dans tout le contenu utilisateur
- Coordonnées centralisées dans `src/lib/constants.ts`
